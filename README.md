# Drop.TN - Générateur de Factures

Générateur de factures professionnel et local pour la marque **Drop.TN**. Créez des factures PDF prêtes à l'envoi pour vos clients en Tunisie en quelques clics.

![Banner](https://github.com/user-attachments/assets/0aa67016-6eaf-458a-adb2-6e31a0763ed6)

## ✨ Caractéristiques

- **Branding Drop.TN** : Logo et informations de l'entreprise pré-configurés.
- **Gestion des Clients** : Saisie facile des informations clients tunisiens.
- **Calcul Automatique** : Gestion automatique du HT, de la TVA (19%, 13%, 7%, 0%) et des frais de livraison.
- **Catalogue Intégré** : Sélection rapide des articles Caplori ou saisie manuelle.
- **Export PDF** : Génération d'une facture professionnelle au format PDF.
- **Mode Local** : Fonctionne entièrement hors-ligne, aucune clé API requise.

## 🚀 Lancement Local

**Prérequis :** Node.js installé.

1. **Installer les dépendances :**
   ```bash
   npm install
   ```

2. **Lancer l'application :**
   ```bash
   npm run dev
   ```

3. **Ouvrir dans le navigateur :**
   L'application sera disponible sur `http://localhost:3000`.

## 🛠️ Stack Technique

- **Frontend** : React 19, TypeScript, Vite
- **Design** : Tailwind CSS v4, Lucide Icons
- **PDF** : jsPDF, autoTable
